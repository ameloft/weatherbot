# WeatherBot: Ваш Особистий Прогноз Погоди в Telegram

**WeatherBot** — це зручний та інтуїтивно зрозумілий Telegram-бот, який надає **актуальну інформацію про погоду** в будь-якій точці світу. Отримуйте миттєвий доступ до поточної погоди та прогнозу на 3 дні, просто надіславши назву міста або координати!

### Можливості

* **Миттєва погода за назвою міста:** Просто введіть назву міста (наприклад, "Kyiv" або "Лондон"), і бот надасть вам поточну температуру та погодні умови.
* **Прогноз на 3 дні:** Отримайте детальний прогноз на найближчі три дні, включно з максимальною та мінімальною температурою, а також описом погодних явищ.
* **Підтримка координат:** Для максимальної точності ви можете надіслати боту **географічні координати** (наприклад, "50.45, 30.52"), щоб отримати погоду для конкретної локації.
* **Інтерактивне меню:** Після введення місця бот запропонує зручне **inline-меню** для вибору між поточною погодою та прогнозом.
* **Зрозумілі описи погоди:** Коди погоди перетворюються на прості та зрозумілі описи українською мовою.

### Використані технології

* **Python:** Основна мова розробки бота.
* **`python-telegram-bot`:** Бібліотека для взаємодії з Telegram Bot API.
* **Open-Meteo API:** Надійне джерело погодних даних.
* **Nominatim (OpenStreetMap):** Використовується для перетворення назв міст у географічні координати.

### Як використовувати

1.  **Запустіть бота:** Знайдіть WeatherBot у Telegram та натисніть "Start".
2.  **Введіть місце:** Надішліть назву міста (наприклад, `Львів`) або координати (наприклад, `49.83, 24.02`).
3.  **Оберіть дію:** За допомогою кнопок оберіть, чи хочете ви побачити поточну погоду, чи прогноз на 3 дні.
